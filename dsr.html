Automated Daily Stock Report for Petroleum Retail

A transformational digital initiative to revolutionize India's petroleum retail industry by eliminating manual processes, reducing operational inefficiencies, and ensuring regulatory compliance.

Project Info

  • My Role: Lead Developer / System Architect
  • Category: Enterprise Software / Petroleum Sector
  • Core Skills: System Architecture, OCR Automation, Hardware API Integration, AI-Powered Analytics.
  • Key Achievement: Designed a system to reduce manual errors from 40% to <2% across 88,000+ outlets.

Project Overview

The Daily Stock Report Automation System (DSRAS) was designed to solve critical challenges in India's petroleum retail sector, where over 90% of 88,000+ outlets rely on manual, paper-based processes.These legacy systems lead to data entry error rates as high as 40%, significant fuel losses, and major compliance issues. This project introduces a comprehensive digital solution to automate daily operations from fuel dispensing to final reporting.

Key Responsibilities

  • Architected a cloud-native, microservices-based system designed for nationwide deployment across more than 88,000 outlets.
  • Designed and integrated an OCR engine using Google Vision API to automate the processing of handwritten receipts and logs in 9 regional languages.
  • Engineered a comprehensive hardware integration framework to connect with over 120 fuel dispenser brands and 40+ Automatic Tank Gauging (ATG) systems for real-time data capture.
  • Developed a multi-role dashboard and an automated DSR generation engine to provide real-time analytics for attendants, managers, and corporate analysts.
  • Implemented an AI-powered fraud detection module to analyze transaction patterns, prevent revenue leakage, and mitigate financial risks.
  • Ensured the system met complex regulatory requirements, including automated GST compliance and E20 blended fuel mandates.
